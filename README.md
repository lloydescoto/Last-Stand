# Last Stand
A Final Requirement for 62DGAME Course

# Game Play
Defend your final base for the incoming enemies.

# Controls
W,A,S,D - Move Character
Left Mouse-Click - Shoot
Space - Plant turrets, walls, mines
Q - Change Carrying Items
N - Call/Next Wave

1. Pygame
